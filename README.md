# Controle Financeiro

Descrição:
Projeto de controle financeiro desenvolvido em Excel para gestão de clientes, pagamentos e valores em aberto. Inclui classificação automática de status (em aberto e em atraso), cálculo de indicadores (total geral, média de pagamentos, valores em atraso) e dashboard interativo com visualização por cliente. Ideal para acompanhamento financeiro simples e eficiente.
